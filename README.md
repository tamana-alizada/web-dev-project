# SweetSaver

## project description
SweetSaver is a multi-vendor web platform designed to reduce food waste while helping users save money. Bakeries and dessert shops can upload their unsold or leftover products at discounted prices, especially towards the end of the day. Customers can browse available offers, reserve items directly through the application, and collect them from the store before the specified pickup deadline.

The system supports multiple shops, each managing their own products and offers independently, while users have personalized accounts where they can track their cart, reservations, and purchase history. The platform provides a smooth and intuitive user experience through features such as category-based browsing, search functionality, and real-time updates of product availability.

SweetSaver is built using Angular for the frontend and Django REST Framework (DRF) for the backend, implementing secure JWT-based authentication. The application follows a full-stack architecture where the frontend communicates with RESTful APIs to perform operations such as user authentication, product browsing, cart management, and order processing.

The main goal of this project is not only to demonstrate modern web development practices (including API integration, authentication, and component-based UI design), but also to contribute to a meaningful real-world problem — minimizing food waste while making quality products more accessible and affordable.
